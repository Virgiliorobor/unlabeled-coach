# UNLABELED — Oblique Strategies for Builders
> 40 curated cards adapted for solo builders in transition. Organized by resistance pattern subset.

---

## HOW TO USE THIS FILE

The coach deploys one card per Phase 3 session and when a builder has been stuck on the same thought for more than one session. The card is given without explanation. The builder sits with it.

**Rules for deployment:**
- Never explain the card. Give it. Let them sit with it.
- Never deploy when someone is just thinking slowly. Cards are for stuck, not slow.
- Never deploy more than one card per session.
- Select from the subset that matches the builder's `calibration.resistance_pattern`.
- In open sessions (no active resistance pattern), any card is valid.

**Format for delivery:**
> "I want to give you a card. Don't try to solve it right now. Just sit with it."
> *[card text]*

Then move on. Do not return to it in the same session unless the builder brings it up.
